# 🌐 Personal Profile Website

This is my personal profile website built using HTML and CSS. It showcases my introduction, skills, education, goals, hobbies, and contact information.

## 📌 About the Project

This project is part of my learning journey in web development. I created this website to practice HTML structure and CSS styling while building my own personal profile page.

## 🚀 Features

* 👨‍💻 About Me section
* 🛠️ Skills list
* 🎯 Goals and ambitions
* 🎮 Hobbies section
* 🎓 Education details (with table)
* 🔗 Social media links
* 📬 Contact form
* 📱 Responsive navigation bar
* ✨ Smooth scrolling

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, hover effects, styling)

## 📂 Project Structure

* `index.html` → Main webpage
* `css/style.css` → Styling file

## ▶️ How to Run

1. Download or clone this repository
2. Open the project folder
3. Open `index.html` in any browser
